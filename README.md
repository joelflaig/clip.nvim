# clip.nvim

**A list-like view for all your vim registers, written in lua.**

## Prerequisites

- `neovim`

## Installation

Using `packer.nvim`:

```lua
use "joelflaig/clip.nvim"
```

Using `lazy.nvim`:

```lua
{
  "joelflaig/clip.nvim",
  event = "VimEnter",
  opts = {
    -- configuration options
  }
}
```
